Overview

The CNISDKPaymentKitConfiguration represents the local configuration of the SDK to use with. These configurations can be stored, but cannot be safely changed once the PaymentKit is initialized. Use this object to construct a configuration for the PaymentKit in your application, and pass it to the CNISDKKitBundle
