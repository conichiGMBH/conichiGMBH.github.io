Overview

The CNISDKPermissionsManager class contains methods that handle asking for iOS permissions for bluetooth, background location and notifications. CNISDKPermissionsManager is deprecated, use CNISDKLocationPermissionManager, CNISDKBluetoothPermissionManager and CNISDKRemoteNotificationsPermissionManager instead.
